# Support

Use GitHub Issues for reproducible bugs and feature requests. Include the output of `syncordactl status --json`, macOS version, source app, output transport, and a concise description of what you expected.

For a security issue, follow [SECURITY.md](SECURITY.md). For a private conduct concern, follow [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

Syncorda is alpha software. The project does not currently offer paid support or uptime guarantees.
